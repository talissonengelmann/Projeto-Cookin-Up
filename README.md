# 🍳 Cookin Up

Projeto desenvolvido durante os estudos de Front-End, criando uma aplicação de busca e organização de receitas.

## 🚀 Acesse o projeto

🔗 [Clique aqui para visualizar o site](https://talissonengelmann.github.io/Projeto-Cookin-Up/)

## 🛠 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## 📚 Sobre o projeto

O Cookin Up é uma aplicação voltada para busca de receitas, desenvolvida com foco em uma experiência simples e intuitiva para encontrar opções de pratos.

Durante o desenvolvimento foram aplicados conceitos importantes de Front-End, como:

- Estruturação semântica com HTML
- Estilização com CSS
- Manipulação de elementos com JavaScript
- Organização de componentes
- Layout responsivo
- Boas práticas de desenvolvimento

## ✨ Funcionalidades

O projeto apresenta:

- 🔎 Busca de receitas
- 🥗 Organização por categorias
- 📱 Interface adaptável para diferentes telas
- 🎨 Layout moderno e responsivo

## 📱 Responsividade

O projeto foi desenvolvido pensando em diferentes dispositivos:

- Desktop
- Tablet
- Mobile
